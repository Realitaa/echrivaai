Design a modern, clean, and professional landing page for a SaaS product called "EchrivaAI".

EchrivaAI is an AI-powered essay correction platform designed for education (schools, teachers, and students). It allows teachers to create assignments, students to submit essays, and the system automatically evaluates them using AI with detailed feedback and scoring.

Tech context (important for design direction):
- Built with Laravel + Inertia.js
- UI components based on shadcn-vue (clean, minimal, component-driven design). You can use existing shadcn-vue components or add any if you want. You are freely to change the components code to match our theme. 
- Focus on modern SaaS UI with smooth UX and reusable components

Target users:
- Teachers who want to automate grading
- Students who want instant feedback
- Schools that need scalable assessment systems

Design style:
- Minimalist, modern SaaS
- Soft gradient background (light purple like #EDE9FE)
- Clean typography, lots of whitespace
- Subtle shadows and rounded cards
- Inspired by Stripe / Linear / Vercel style

Sections to include:

1. Hero Section
- Headline: "AI-Powered Essay Feedback in Seconds"
- Subheadline: emphasize speed, accuracy, and learning improvement
- CTA buttons: "Get Started" and "See Demo"
- Illustration: dashboard preview or AI analyzing essay

2. Problem → Solution Section
- Problem: manual grading is slow, inconsistent, exhausting
- Solution: EchrivaAI automates scoring + gives structured feedback

3. Features Section (important)
Show key features based on real system:
- AI Essay Evaluation (auto scoring + feedback)
- Multiple Submission Attempts (students can improve essays)
- Teacher Review & Final Score (AI + human collaboration)
- Classroom & Task Management (teachers manage assignments easily)
- File Upload Support (essay files attached to submissions)
- AI Feedback History (track improvements over time)

4. How It Works (3 steps)
- Teacher creates task
- Student submits essay
- AI evaluates and gives feedback instantly

5. Dashboard Preview Section
- Show UI mockup: submission list, feedback panel, scoring system

6. Benefits Section
- Save time for teachers
- Faster feedback for students
- Consistent and objective grading

7. Testimonials / Trust Section (placeholder content)

8. Pricing Section (optional placeholder)

9. CTA Section
- Strong call to action: "Start improving writing with AI today"

10. Footer
- Simple links, branding, minimal

UI Component Notes:
- Use card-based layout (shadcn style)
- Use reusable components (buttons, badges, alerts)
- Keep spacing consistent and scalable
- Avoid overly decorative elements

Tone:
- Professional, slightly futuristic (AI-focused)
- Trustworthy and educational

Goal:
The landing page should feel like a real SaaS product, not a template. It should communicate clearly what the product does and why it is valuable.
